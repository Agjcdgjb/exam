# 資訊安全管理概論測驗系統

這是一個基於 Flask 的線上測驗系統，用於測試資訊安全管理概論的知識。

## 功能特點

- 隨機選擇 25 題進行測驗
- 即時評分系統
- 答題結果分析
- 響應式設計

## 本地運行

1. 安裝依賴：
```bash
pip install -r requirements.txt
```

2. 運行應用：
```bash
python quiz_app.py
```

3. 在瀏覽器中訪問：`http://localhost:5000`
