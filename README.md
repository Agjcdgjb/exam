# 資訊安全管理概論測驗系統

這是一個純前端的線上測驗系統，用於測試資訊安全管理概論的知識。

## 功能特點

- 隨機選擇 25 題進行測驗
- 即時評分系統
- 答題結果分析
- 響應式設計
- 無需後端服務器

## 使用方式

1. 直接訪問 GitHub Pages 網址：
https://agjcdgjb.github.io/quiz-system/

2. 或下載後在本地運行：
   - 下載所有文件
   - 使用任何靜態文件服務器（如 Python 的 `http.server`）運行
   - 在瀏覽器中訪問 `http://localhost:8000`

## 技術棧

- HTML5
- CSS3
- JavaScript (ES6+)
- JSON 數據存儲
